4.flex-basis 彈性基準係數

flex-basis 是指在 row 或 column 軸向時，彈性項目的基準尺寸(長度)

flex-direction: row 時，flex-basis 基準為 width，預設情況 flex-basis:auto，width可以被設定

flex-direction: column 時，flex-basis 基準為 height，預設情況 flex-basis:auto，height可以被設定

但是當 flex-basis 設定數值時，優先權會比 width 或 height 高

flex-basis:0 時，長度失效，width或height設定再大也無法動作
